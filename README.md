##   Required Libraries in python
1.) open the code through python and open Python terminal and execute the below commands.
2.) pip install streamlit 
3.) pip install pandas
4.) pip install numpy
5.) pip install scikit-learn

After installing successfully open command prompt and change the directory to the path where the AIPMCode is present by cd <path> and press enter
